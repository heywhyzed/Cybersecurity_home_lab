# Cybersecurity Home Lab
## Project OverviewThis project documents the setup of my personal cybersecurity laboratory used for learning ethical hacking, network analysis, and incident response.
## Objectives 
- Build a cybersecurity practice environment
- Install Kali Linux virtual machine
- Practice security tools safely
- Develop hands-on cybersecurity skills
  
## Lab EnvironmentHost 
System: Windows Laptop  
Virtualization Software: VirtualBox  
Operating System: Kali Linux

## Installation Steps
1. Installed VirtualBox.
2. Created new virtual machine.
3. Installed Kali Linux OS.
4. Configured virtual networking.
5. Verified internet connectivity.
